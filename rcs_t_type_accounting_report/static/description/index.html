Why T-Type Accounting Report?

Enhance your Odoo accounting with the T-Type Accounting Report module, which delivers a classic T-account view for both the Balance Sheet and Profit & Loss reports. This format mirrors traditional bookkeeping structures, where the Balance Sheet displays Assets on the left (debit side) and Liabilities & Equity on the right (credit side). For the Profit & Loss report, Revenues are shown on the right side (credit), while Expenses appear on the left side (debit), offering a clear and familiar layout for accountants and finance professionals.

Whether you're migrating from other accounting platforms like QuickBooks, generating dynamic balance sheet reports in Odoo, or delivering accounting training using Odoo's financial tools, this module offers a clean and intuitive way to visualize financial data.

Set Up

To begin with, have the T-Type Accounting Report for Odoo downloaded in your system. For its proper setup, go through the below steps.

Firstly, you need to install the T-Type Accounting Report for Odoo App from Odoo Apps. It's easy to locate, just search on the name, you can identify it with the T-type Accounting icon. Hold on, you also need to enable it.

Apps >> T-Type Accounting Report for Odoo >> Install

Work Flow for Profit & Loss T-type Report

Ensure that odoo accounting is set up with a chart of accounts, journals, fiscal year and periods, and that journal entries are posted.

To access the T-Type Report, navigate to Invoicing >> Reporting >> Profit & Loss T-Type Report, and click on it
You can view the current financial year's Profit and Loss report by default.
You can also apply filters within the report. For example, if you want to create the Profit & Loss report for a specific date range, you can do so. Similarly, to generate the report for a particular company, simply select the company and click the 'Apply' button.
When you click the 'Apply' button, the report is generated based on the selected filters. It displays Income on the left and Expenses on the right, grouped by account, with subtotals and running balances (if applicable).

Work Flow for Balance Sheet T-type Report

To access the T-Type Report, navigate to Invoicing >> Reporting >> Balance Sheet T-Type Report, and click on it
You can view the current financial year's Balance Sheet t-type report by default.
You can also apply filters within the report. For example, if you want to generate the Balance sheet report for a specific date range, you can do so. Similarly, if you want to generate the report for a particular company, simply select the company and click the 'Apply' button to generate the report.
When you click the 'Apply' button, the report is generated based on the selected filters. It displays Equity and Liabilities and related accounts (Debits) on the left, and Assets, Current Assets and related accounts (Credits) on the right, grouped by account. The report includes subtotals and running balances, where applicable.
You can also export the generated report as an Excel file. You'll see the 'Export (XLSX)' button, and when you click it, the report will be downloaded in Excel format.
